Sobre a linguagem

Rust é uma linguagem extremamente rápida de tipagem estática que garante operações memory safety e concorrência sem condições de corrida.

Legal mas:

  • Eu ja programo em ruby
  • Ruby paga minhas contas
  • Eu gosto de ruby

Porque devo me preocupar com Rust?

"A language that doesn’t affect the way you think about programming is not worth knowing."

- Alan Perlis

O que eu ganho?

No minimo, podemos aprender um bocado de boas práticas com uma linguagem que prove Concorrência, Thread-safe, Imutabilidade by default e é extremamente rápida.

Sem perder a sintaxe agradável e as facilidades de uma linguagem dinâmica.
