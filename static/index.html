Semelhanças

  • Convenção para nome de variáveis 
    let snake_case
  • Blocks com '{}' 
    while true { do_stuff() }
  • Parece dinamicamente tipada, mas é inferência. 
    let num = 0; let string = "a"
  • Não precisa de 'return' no fim do bloco.

Diferenças

  • O statement deve terminar com ';' menos o último.
  • 'fn' no lugar de 'def'
  • Imutabilidade by default ou mutabilidade explicita!
  • Não tente enganar o compilador! Ele vai reclamar.
    • fn f(x){
  let a = 0; // Imutable
  let mut b = 1; // Mutable
  b = a + 1; // Permitido
  a = b + 1; // Erro! 

  a + b + x // retorna o ultimo statement
}
